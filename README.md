# Secure Comms Simulator

Educational web game for learning cryptography concepts (Rail Fence Cipher, Diffie-Hellman, MITM).

## Setup Instructions

If you just cloned this repository, follow these steps to run the app:

1.  **Install Dependencies**
    Open your terminal in this folder and run:
    ```bash
    npm install
    ```

2.  **Start the Server**
    Run the development server:
    ```bash
    npm run dev
    ```

3.  **Open the App**
    Click the link shown in the terminal (usually `http://localhost:5173`) to view the app in your browser.
